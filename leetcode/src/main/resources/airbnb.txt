Palindrome Pairs 面经

给你一个String, 问里面能形成最少palindrome substring的个数。
For example
“aaa” -> 最少就是1个 就是aaa
“aaab” -> 最少就是2个 就是aaa和b
“abc” -> 最少就是3个 就是 a，b和c
